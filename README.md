# Building from source
### Requirements
	- Linux-based system
	- GCC compiler
	- The [Webtoolkit library](https://www.webtoolkit.eu/wt)
	- Make
### Compiling
	1. mkdir <folder-name>
	2. cd <folder-name>
	3. git clone https://github.com/gergoszathmari/parser.git
	4. cd parser
	5. make

### Running
	* Run the "run.sh" script found in the root directory
